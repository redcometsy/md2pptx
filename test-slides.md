# Markdown 转 PPT 效果测试

## 项目背景
- 技术栈: Python + React + PostgreSQL
- 目标用户: 企业数据分析师
- 核心需求: 将数据分析结果快速生成演示文稿
- 上线计划: 2026年Q3

## 核心功能
- Markdown 自动解析为幻灯片
- 支持标题、列表、代码块
- 暗色主题，16:9 宽屏比例
- 输出标准 .pptx 文件

### 技术细节
- python-pptx 操作 PowerPoint 对象模型
- 正则解析 Markdown 语法树
- 每页自动编号，绿色强调色

## 代码示例
- 解析器核心逻辑约 80 行
- 支持 H1/H2/H3 三级标题
- 支持无序列表和正文段落
- 支持代码块标记（跳过渲染）

## 效果对比

### 优势
- 纯文本编写，版本控制友好
- 样式统一，无需手动排版
- 批量生成，适合自动化场景

### 局限
- 不支持图片/表格（可扩展）
- 不支持自定义主题切换
- 不支持演讲者备注

## 后续计划
- 添加图片支持（![]() 语法）

- 添加多主题切换（浅色/深色/品牌色）
