# Markdown to PPTX Converter

Python script that converts Markdown files to PowerPoint (.pptx) presentations.

## Features
- Dark theme, 16:9 widescreen
- H1 = title slide, H2 = new slide, H3 = subsection
- Bullet lists, paragraph text
- Auto page numbering
- Green accent color

## Usage
```bash
python md2pptx.py input.md [output.pptx]
```

## Requirements
- Python 3.7+
- python-pptx (`pip install python-pptx`)
